Le back-end¶

Le terme back-end désigne la partie du code associée à Django et écrite en python. Le but du back-end est de récupérer, générer et combiner les différentes données du site afin de les envoyer au font-end.

Vous trouverez dans les liens ci-dessous une documentation spécifique à chacun d’entre eux. Vous pouvez également retrouver la documentation technique des modules ici.
